## 🏃 Instalação

```bash

    // Clonar repositório
    $ git clone https://github.com/BlvkRoot/facile-challenge

    // Acessar diretório
    $ cd facile-challenge

    // Instalar dependências
    $ yarn

    // Iniciar projeto
    $ yarn dev
```
